Dieser Ordner enthält:

- die Python Datei "external_solver.py" in welcher die Basisklasse "ExternalSolver" definiert ist. Diese enthält alle Methoden, welche für die Schnittstelle implementiert werden müssen, mit umfangreichen Kommentaren über die Funktionalität jeder einzelner Methode.

- den Unterordner "test_examples/01_square/" mit einem Dummy Optimierungsbeispiel, in welchem die Form einer initial quadratischen Platte verändert wird, indem dem Mittelpunkt eine senkrechte Sensitivität zugewiesen wird.
