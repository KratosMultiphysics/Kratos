MPE_ReturnTags

 Returns tags allocated with MPE_GetTags.

Synopsis

int MPE_ReturnTags( comm, first_tag, ntags )
MPI_Comm comm;
int      first_tag, ntags;

Input Parameters

comm
Communicator to return tags to
first_tag
First of the tags to return
ntags
Number of tags to return.

MPE_GetTags
