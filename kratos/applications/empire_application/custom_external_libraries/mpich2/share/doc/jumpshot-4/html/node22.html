3.4.1.4.3 Time Info Box

Figure 3.20: Time Info Box displays the time of where it pops up.

Time Info Box is created by right clicking in the empty space in either the timeline or the time ruler canvas, as in Figure 3.20. This Info Box is usually used as a marker for a single event in time.
