This norm is only applicable to `Matrix` type variables.

<p align="center">$$ ||\underline{A}||_{lpq} = \left(\sum_{j=0}^{n-1}\left(\sum_{i=0}^{n-1}|a_{ij}|^p \right )^{q/p} \right )^{1/q}$$</p>
